# 🛒 Retail Data Analysis Project

## 📌 Overview

This project analyzes a retail sales dataset using **SQL and Python** to understand sales trends, customer behavior, and regional performance.

---

## 🎯 Objectives

* Analyze sales by category and region
* Identify top-performing cities
* Understand customer shipping preferences
* Visualize data using graphs

---

## 🛠️ Tools & Technologies

* Python
* SQLite (SQL)
* Pandas
* Matplotlib
* Visual Studio Code

---

## 📂 Project Structure

* `train.csv` → Dataset file
* `retail.db` → SQLite database
* `import_data.py` → Script to import CSV into database
* `queries.sql` → SQL queries for analysis
* `graphs.py` → Python code for data visualization

---

## ⚙️ How to Run the Project

### 1. Import Data into Database

Run:

```bash
python import_data.py
```

### 2. Run SQL Queries

Open SQLite extension in VS Code and execute queries from `queries.sql`.

### 3. Generate Graphs

Run:

```bash
python graphs.py
```
